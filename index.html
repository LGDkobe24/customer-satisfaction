Performing Sentiment Analysis of tweets to various brands

I am extracting tweets to various e-commerce companies in India like Amazon, Flipkart, etc and performing sentiment analysis on them to figure out the customer satisfaction level of each company. Currently, I'm using http://text-processing.com's API to perform the sentiment analysis. This will be changed in subsequent versions to use my own algorithm.

The tweets collected are taken over a period of 1 week, again, this will be expanded to include a much larger data set soon.

And finally, the result which I got.

Surprisingly, Myntra performs poorly, and Snapdeal looks to be way better than the giants like Amazon and Flipkart!

Further more analysis will be added. The codes taken to extract, calculate and plot the data, along with the tweets and the result is provided in the project file.
